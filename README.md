\# Gestor Finanzas V3



Aplicación desarrollada en Java para la gestión de finanzas personales.



\## Funcionalidades actuales

\- Registro de ingresos

\- Registro de gastos

\- Cálculo de balance

\- Almacenamiento en MySQL

\- Historial en fichero `.txt`

\- Listado de ingresos y gastos desde base de datos



\## Tecnologías utilizadas

\- Java

\- Eclipse

\- MySQL

\- JDBC

\- Git \& GitHub



\## Objetivo del proyecto

Evolucionar esta aplicación hacia una herramienta más completa con:

\- Gestión mensual

\- Ahorro acumulado

\- Objetivos financieros (coche, casa, viajes)

\- Interfaz gráfica con JavaFX



\## Estado actual

Versión 3 (consola + base de datos + ficheros)
